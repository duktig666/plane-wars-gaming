# plane-wars-gaming

##  简介

> 基于js+html+css实现的飞机大战小游戏。
>
> 本项目为曾经学习js所做作品，用时两周。

### 游戏特色

- 独特的飞机大战背景音效和子弹撞击飞机的音效
- 不同的增益Buff和减益Buff
- 开局默认3条生命，可吃小飞机Buff增加生命
- 时间、分数统计
- 暂停游戏
- 终极Boss

### Boss说明

Boss1——黄金狮王

- 60秒时出现，一定频率上下移动，发射的子弹碰撞屏幕会反弹
- 每出现一次血量+20
- 每出现一次，一次发射子弹数+1

![黄金狮王](https://cos.duktig.cn/typora/20210818100647.png)
